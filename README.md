# SketchFlow

SketchFlow is a simple whiteboard for teams and students. It gives you an
infinite canvas for diagrams, notes, sketches, planning, and visual thinking.

This project is based on the open-source Excalidraw editor and is customized
with SketchFlow branding.

## Features

- Infinite canvas whiteboard
- Hand-drawn drawing style
- Shapes, arrows, text, images, and freehand drawing
- Browser-only local saving
- Export to image and file
- Light and dark themes
- SketchFlow + premium placeholders for future cloud features
- Progressive web app support

## Run Locally

Install dependencies:

```powershell
yarn
```

Start the app:

```powershell
yarn start
```

Open the local URL shown in the terminal. It is usually:

```text
http://localhost:3001
```

## Build

Create a production build:

```powershell
yarn build
```

The generated app is placed in:

```text
excalidraw-app/build
```

## Current Storage

SketchFlow currently saves drawings in the browser and supports exporting files.
Cloud saving, accounts, payments, and SketchFlow + features are planned for a
future backend.
